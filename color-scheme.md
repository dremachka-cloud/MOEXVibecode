# Цветовая схема сайта

## Основные цвета

### Черный фон
- `#1b1b1b` - Основной черный фон страницы
- `#252525` - Темно-серый для карточек
- `#2f2f2f` - Светло-серый для поднятых элементов

### Акцентные цвета
- `#5CDB95` - **Яркий зеленый** (Primary)
  - Используется для: основных кнопок, акцентов, календарных меток заседаний

- `#8EE4AF` - **Светлый мятный** (Primary Light)
  - Используется для: ховеров, успешных уведомлений, светлых акцентов

- `#379683` - **Темный зелено-бирюзовый** (Primary Dark)
  - Используется для: навигации, границ календаря, подчеркиваний

- `#907163` - **Коричневый** (Secondary)
  - Используется для: вторичных элементов, предупреждений

### Текстовые цвета
- `#EDF5E1` - **Светлый бежево-зеленый** (Text Light)
  - Основной светлый текст, заголовки

- `#c5c1b9` - **Серый** (Text Gray)
  - Обычный текст, описания

- `#8a8780` - **Приглушенный серый** (Text Muted)
  - Вспомогательный текст, метаданные

### Служебные цвета
- `#8EE4AF` - Успех
- `#907163` - Предупреждение
- `#d47a7a` - Опасность

## Применение в компонентах

### Навигация
- Фон активной ссылки: `rgba(92, 219, 149, 0.15)`
- Цвет активной ссылки: `#5CDB95`

### Календарь
- Заголовки дней недели: фон `rgba(55, 150, 131, 0.15)`, граница `#379683`
- День с заседанием: граница `#379683`
- Бейдж заседания: фон `#5CDB95`, текст `#1b1b1b`
- Заголовок месяца: левая граница `#5CDB95`

### Карточки комитетов
- Граница при ховере: `rgba(92, 219, 149, 0.3)`
- Фон при клике: `rgba(92, 219, 149, 0.12)`
